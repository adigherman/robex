# robex Package:

The goal of `robex` is to provide Robust Brain Extraction (‘ROBEX’) in a
wrapper format.

## Installation

You can install `robex` from GitHub with:

``` r
# install.packages("remotes")
remotes::install_github("muschellij2/robex")
```
